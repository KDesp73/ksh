# ksh

My shell

## Features

- Exports
- Aliases
- History
- Sourcing
- Interactive UI
- Builtin commands (cd, echo, etc)

## Get Started

```bash
# 0. Clone
$ git clone https://github.com/KDesp73/ksh --depth=1
$ cd ksh

# 1. Build
$ make type=RELEASE

# 2. Install
$ make install
```
